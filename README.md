## ğŸ› ï¸ How to Use

### Screen Mirroring (Wireless)
1. Make sure your PC and Phone are on the **Same Wi-Fi Network**.
2. Find your Wireless Debugging IP & Port from Developer Options (Android 11+) or use the Magic USB Trick (Android 10 & below).
3. Enter the IP and Port in the Mobile Link dashboard.
4. Click **Connect Wirelessly**.

### FTP File Transfer
1. Install a lightweight app like **WiFi FTP Server** on your Android phone and tap 'Start'.
2. Click the **Open Phone Storage** button on the Mobile Link dashboard.
3. Enter the Port number shown in your mobile app.
4. Your phone storage will instantly open in Windows File Explorer!
